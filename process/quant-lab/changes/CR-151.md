## 范围

### In Scope

1. 多因子策略本地 E2E 统计准入框架。
2. FDR / multiple testing gate。
3. Newey-West 或等价自相关稳健统计 gate。
4. Walk-forward / OOS fold manifest。
5. PBO / DSR 或 fixture-friendly overfit risk report。
6. `StrategyAdmissionStatisticalGate` 与现有 `StrategyAdmissionPackage` / `BacktestRunSpec` / `ResearchDatasetSpec` 的 linkage。
7. Fixture / static-only validation 证明 gate 语义。
8. CP2 范围门、CP3 设计门、CP5 设计证据批次、CP6/CP7 实现验证、CP8 release readiness。

### Deferred / Wave B Candidate

1. Factor correlation clustering。
2. Capacity / impact report。
3. IR / tracking error / Active Share。
4. PIT universe audit。
5. Benchmark-relative / attribution / regime analysis。
6. Extended factor evaluation diagnostics: IC decay by lag, half-life, turnover, liquidity/capacity view, orthogonalization against known risk factors, monotonicity and quantile spread。

这些能力对生产质量重要，但 CP2/CP3 可决定是否留在 CR151 Wave B、拆为 CR151.5，或移动到 CR154 横切生产可靠性门。
